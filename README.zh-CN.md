## 这是什么？

一组个人开发套件，旨在展示各个场景下的最佳实践。



## 使用

下载/克隆仓库到本地；

安装 ``xcodegen``，``cd`` 到 ``Example`` 目录下，运行 ``xcodegen``；

运行 ``pod install``。

采用 cocoapods 结构，但尚未发布。



请通读源码。
